# rssynthesis
## Pre-Requisites
- Python 3.11 or higher (use pyenv)
- Node 20 or highey (use nvm)

## To Install
```bash
make install
```

## To run locally:
In one term start tailwind by doing `make tw`. Then, in other start the main app by doing `make run`.

If you want to run via docker-compose:
```bash
docker compose up
```
